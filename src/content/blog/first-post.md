# First Post

This is a sample blog post written in Markdown. The blog system supports frontmatter metadata and renders Markdown content.

## Features

- Markdown syntax support
- Frontmatter metadata
- Individual post pages
- Blog index listing

You can write your blog posts using standard Markdown syntax, including:

- **Bold text**
- *Italic text*
- `Code snippets`
- Lists (ordered and unordered)
